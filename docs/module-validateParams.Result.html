    validateParams. Result

    Fcnew Result(parameterList, constraintsList-, options, validateAttributes, validateConstraints, errorDetailsopt, errorsopt)

    A prototype for objects representing the results of validating a list of parameters against a list of constraints. Instances of this prototype should only be constructed internally within the validateParams.validate() function.

    Since:
    • version 1.1.1
    See:
    • module:validateParams.validate
    • external:validate
    Parameters:
    Name Type Attributes Description
    parameterList Arguments | Array

    the parameter list that was validated.
    constraintsList- Array.<ParameterConstraints>

    the parameter constraints applied by the validation
    options Object

    the options passed to the validate() function from validate.js.
    validateAttributes Object

    the attributes data structure passed to the validate() function from validate.js.
    validateConstraints Object.<string, AttributeContraints>

    the constraints object passed to the validate() function from validate.js.
    errorDetails Object <optional>

    the error details returned from the validate() function from validate.js with the format option set to detailed.
    errors * <optional>

    the errors returned from the validate() function from validate.js with the requested formatting applied.
    Throws:

    A type error is thrown if any of the parameters are missing or invalid.

    Type
    TypeError
